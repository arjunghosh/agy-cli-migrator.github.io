100% Automated Backups

Automatically creates timestamped settings backup files (`settings.json.YYYYMMDD_HHMMSS.bak`) before executing any file operations.

⚙️

Windows Junction Support

Full cross-platform compliance. Automatically uses standard POSIX symlinks on Linux/macOS and falls back to Local Directory Junctions on Windows without administrative elevation.

⏪

One-Command Rollback

Run `--rollback` at any time to instantly unlink all migrated skills and restore your settings file from the latest backup.
